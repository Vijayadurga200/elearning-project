# 🧩 EduAccess - Disability-Aware Adaptive E-Learning Platform

A fully accessible e-learning platform with AI-powered assistive technologies for visually and hearing-impaired learners.

## ✨ Features

### 👁️ **Visual Impairment Support**
- 🎤 Voice command navigation (Speech-to-Text)
- 🔊 Real-time Text-to-Speech (TTS)
- ♿ Screen reader optimized (ARIA labels, semantic HTML)
- ⌨️ Full keyboard navigation

### 👂 **Hearing Impairment Support**
- 📝 Automatic subtitle generation (Whisper AI)
- 🤟 Sign language animations
- 👀 Visual notifications & cues
- 📄 Full transcripts & summaries

### 📊 **Admin Features**
- PowerBI analytics dashboard
- Usage heatmaps
- Accessibility feature tracking
- Multi-language support
